Open-source, low-cost biofloc RAS water-quality node. Measures pH, Electrical conductivity, Dissolved oxygen, Oxidation-reduction potential, turbidity, and temperature; uploads via Blues Notecard/Notehub. Computes TSS on-device every 10 minutes using two regressions (simple: turbidity only; multiple: turbidity + EC + temperature). Coefficients are dataset-specific; re-fit for your site and update in firmware or via serial. The repository includes: Arduino code with TSS prediction, hardware (wiring diagrams and build notes), and Bill of Materials. Outputs raw sensors and TSS predictions to Notehub JSON for dashboards and alerts.
